Using torba on OSX

Em Português

A port to OSX was created to make OSX useful for UNIX people. Following are the compilation and installation steps.

Compile torba from the osx directory of the snapshot: 

cd osx
make

To compile torba on OSX 10.5 edit the Makefile and replace: 

libtwmhack.so.$(LVERS): twm_hack.so
	$(CC) -shared -fpic -o libtwmhack.so.$(LVERS) $(LDADD) twm_hack.so
With: 
libtwmhack.so.$(LVERS): twm_hack.so
	$(CC) -shared -bundle -fpic -o libtwmhack.so.$(LVERS) $(LDADD) twm_hack.so

Install torba: 

sudo make install

Download dmenu and compile it: 
make
Install dmenu: 
sudo make install
Create a .xinitrc in the home dir of the user and add at least the following line: 
exec torba
Set the X11 preferences of OSX to the following:





This all results in torba bliss!
